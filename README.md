## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Usage

1. **Select a Sprint**: Use the dropdown to filter tasks by sprint
2. **Filter by Date**: Choose a specific date to see tasks created on that day
3. **View Metrics**: See sprint velocity, completion rates, and progress
4. **Generate Presentation**: Click "Generate Sprint Review Presentation" to create a PDF deck for your business team
